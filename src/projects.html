Gasoline Alley

This project is a demo for a game that I've been conceptualizing since my time in university. Inspired by the classic Resident Evil games, players will find themselves stranded in a remote town deep in the mountains where the inhabitants have become a frenzied horde of blood-thirsty monsters. The gameplay centers around exploring the eerie locale and solving puzzles with the ultimate goal of finding a way to escape. Along the way, players will have to defend themselves from the town's residents, all while managing limited resources.

This project is being developed in Unreal Engine 5. The core mechanics are still being implemented, but once it is in a shareable state I'll post a link here!
